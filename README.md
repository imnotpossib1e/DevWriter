## 프로젝트 개요
DevWriterAI는 개발자를 위한 기술 블로그 생성기입니다.<br/>
OpenAI API를 활용해 주제를 입력하면 몇 초 만에 완성도 높은 마크다운 기술 블로그를 만들어줍니다.
<br/>

## 개발 기간
> 2026.02.02 ~ 2026.02.27
<br/>

## 주요 기능
  ### 📝 기술 블로그 생성 <br/>
  주제·키워드·템플릿·길이·톤 선택 → AI 자동 완성

  ### 💾 Export / Copy <br/>
  HTML, Markdown으로 저장 및 클립보드 복사

  ### 🕒 블로그 생성 히스토리 <br/>
  최대 10개 히스토리 저장 및 관리
<br/>
